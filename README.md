## Getting Started

Before starting the application, run command below to scaffold local sqlite database.

```bash
npm run migrate:db
```

After that, run the development server:

```bash
npm run dev
```

> Sign in with Google require update .env file GOOGLE_CLIENT_ID & GOOGLE_CLIENT_SECRET
